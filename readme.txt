A HTML5 website using JavaScript and JQuery to play games.

1. Match the Brand:
The player drags the automotive brand logo to the matching company name using the mouse. Ages 5 and up.

2. Autocross:
The player uses keyboard inputs to move a car on a track, keeping it between pairs of orange cones. Ages 8 and up.

3. Auto Trivia:
The player answers multiple-choice automotive themed trivia questions. Ages 13 and up.
